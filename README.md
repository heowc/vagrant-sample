# vagrant-sample

```bash
$ cd {path}
$ vagrant up

Success...!

$ vagrant ssh
```

## box

```bash
$ vagrant box add centos/6
```

## plugin

Optional: VirtualBox

```bash
$ vagrant plugin install vagrant-vbguest
```

Optional: windows OS

```bash
$ vagrant plugin install vagrant-winnfsd
```

Optional: docker-compose in box

```bash
$ vagrant plugin install vagrant-docker-compose
```
